# Spotify AI

This project should initially have 3 functions:

- User describes their mood → AI analyzes it → makes a call to Spotify for music recommendations ✅
- User uploads a photo → AI analyzes it → makes a call to Spotify for music recommendations ✅
- User uploads an album cover → AI analyzes it and tells more about the artist and the history of that album ✅

Technologies that must be learned in this project:
- AI calling ✅
- RAG ⏳
- Memory ⏳
- Embeddings ⏳
- Vision ⏳
- Agents ⏳
